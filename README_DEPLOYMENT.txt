🌐 PRODUCTION DEPLOYMENT (3 STEPS)

STEP 1: Push to GitHub
  git pull origin main --allow-unrelated-histories --no-edit
  git push -u origin main

STEP 2: Deploy Backend → HF Spaces (10 min)
  1. Go to: https://huggingface.co/spaces
  2. Create Space: "surakshanet-ai-backend" (Docker SDK)
  3. Add 7 environment secrets from .env file
  4. Auto-deploys with Dockerfile

STEP 3: Deploy Frontend → Vercel (15 min)
  1. Go to: https://vercel.com/login
  2. Import GitHub: Rohithburla51/SURAKSHANET-AI
  3. Set environment: NEXT_PUBLIC_API_URL=<HF-Spaces-Backend-URL>
  4. Deploy

STEP 4: Verify Live
  curl https://username-surakshanet-ai-backend.hf.space/health
  → {"status": "ok", "demo_mode": false}
